##  Editing the file
Edit the required value in `botwaconfig.json`.
```json
{
    "prefix": "!",
    "ownerNumber": ["62895607019922@c.us", "628"],
    "exegtps": "server.exe",
    "player": "players",
    "world": "worlds",
    "playeronline": "playeronline.txt",
    "gemfolder": "gemdb",
    "nameserver": "GTPS"
}
```
